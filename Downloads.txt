<p>In order to play Warbound Ragnarok Online, you need to download the following client files:</p>
<ol>
<li><strong><a title="kRO Client Files" href="https://mega.nz/file/ICw1zbYa#8OdIqPt5K1pya9TLHgA8813J9EbKLDDra1Fbla5tTzo" target="_blank" rel="noopener">kRO Client</a> (<em>1.4 GB</em>) -&nbsp; </strong>Download and extract this folder somewhere. Preferably directly under your <em>C:\</em> drive if using Windows OS. This folder contains a heavily cleaned up set of data files from an official korean 2018-12-22 client. You should not substitute this with any other kRO release as this will result in errors.<br /><br /></li>
<li><strong><a title="WRO Client Files" href="https://mega.nz/file/FfwVgDxD#wk-LaDyUb26C6AbesUb2bN_8CY4RRlipFz-U565WZ6w" target="_blank" rel="noopener">WRO Client</a> (<em>302 MB</em>) - </strong>Download and extract these files into the kRO Client directory. <li><strong><a title="BGM Folder" href="https://mega.nz/file/1GZiFL7I#2sTrMyGdQQnMW8c7ZnmhGcU_EJnmWg7yEVCyz37N57Y" target="_blank" rel="noopener">BGM Folder</a> (<em>306 MB</em>) - </strong>Optional download.
